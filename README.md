# be-the-hero

Projeto criado no evento Semana omnistack 11 realizado pela [Rocketseat](https://rocketseat.com.br).

## Stack

- NodeJS
- React
- React Native

## Iniciar aplicação

Backend:

Intalar as dependências
```
npm i
```
Rodar as migrations do projeto
```
npm run db:latest
```
Start na aplicação
```
npm run dev
```

Frontend:

Intalar as dependências
```
npm i
```
Start na aplicação
```
npm start
```
